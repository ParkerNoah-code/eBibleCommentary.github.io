Ultralearning AI

Week 1: Introduction to LLMs & Transformers

Goals: Understand the basics of LLMs (Large Language Models) and the Transformer architecture.
Assignments:
  • Go through Part 1 of Practical Deep Learning by fast.ai.
  • Read "The Illustrated Transformer" by Jay Alammar.
Deliverables:
  • Write a one-page summary explaining the Transformer architecture, focusing on how attention works.

Week 2: Exploring Models with Fast.ai

Goals: Gain hands-on experience with pre-trained models and perform basic fine-tuning.
Assignments:
  • Go through Part 2 of Practical Deep Learning by fast.ai.
  • Prepare a small text dataset for fine-tuning.
Deliverables:
  • Fine-tune a small pre-trained LLM using Hugging Face on your dataset and implement it in FastAPI.

Week 3: Data Handling & Preprocessing for NLP

Goals: Learn how to clean and preprocess text data for NLP tasks.
Assignments:
  • Read articles on data preprocessing techniques for NLP.
  • Practice cleaning a raw text dataset using Python libraries (Pandas, NLTK, SpaCy).
Deliverables:
  • Create a data preprocessing pipeline and present cleaned data ready for modeling.

Week 4: FastAPI Integration

Goals: Integrate pre-trained LLMs with a FastAPI backend.
Assignments:
  • Read the FastAPI documentation on creating endpoints.
  • Integrate your fine-tuned LLM into FastAPI.
Deliverables:
  • Create an endpoint that uses your fine-tuned LLM to handle text input and return a response.

Week 5: Containerizing Your AI Application

Goals: Learn how to containerize your application using Docker.
Assignments:
  • Complete the Docker "Getting Started" guide.
  • Create a Dockerfile for your FastAPI application.
Deliverables:
  • A Docker container that successfully runs your FastAPI application.

Week 6: Deployment, Monitoring, and Scaling

Goals: Deploy your application to a cloud provider and implement basic monitoring.
Assignments:
  • Deploy your Docker container to AWS or a cloud provider of your choice.
  • Implement basic logging and monitoring of your API.
Deliverables:
  • Provide a public URL to access your deployed API.
  • Set up a simple monitoring dashboard for your API.

Week 7: Basic React + API Communication

Goals: Create a basic React frontend that communicates with your backend API.
Assignments:
  • Read the React documentation on building simple components and making API requests.
Deliverables:
  • A basic React app that can send a request to your FastAPI backend and display the result.

Week 8: Robust API Communication

Goals: Implement advanced API communication features like error handling and retries.
Assignments:
  • Learn about error handling and retry logic in JavaScript.
Deliverables:
  • Implement robust API communication in your React app with error handling and retries.

Weeks 9-10: Project 1 – AI Chatbot with Customer Support Focus

Goals: Build a complete AI chatbot with added features like sentiment analysis and FAQ integration.
Assignments:
  • Design and implement a chatbot using your fine-tuned LLM.
  • Integrate a FAQ dataset and implement sentiment analysis.
Deliverables:
  • A fully functional chatbot with FAQ integration and sentiment analysis features.

Weeks 11-12: Project 2 – AI Content Analysis Dashboard

Goals: Create a comprehensive content analysis dashboard with summarization, sentiment, and keyword extraction.
Assignments:
  • Develop an AI-based content analysis application.
  • Implement text summarization, sentiment analysis, and keyword extraction.
Deliverables:
  • A fully functional web application that performs content analysis on user-provided text.
