Python Resources Walkthrough

Visit python.org to download the Python interpreter.

The Python Tutorial shows various Python examples, and The Python Standard Library provides a full description of Python's built in capabilities. The official repository site for python libraries is pypi.org, although most of the time a google search or another project is the best way to find and learn about Python libraries.

In Python Full Course for free, Bro Code has video tutorials that cover the basics of Python programming as well as example programs that teach how to use them in projects. From freeCodeCamp.org, Harvard CS50’s Introduction to Programming with Python – Full University Course has good lectures on Object Oriented Programming and Unit Testing, and Python for Everybody - Full University Python Course teaches more advanced Python concepts with networks, web services, databases, and data visualization.
